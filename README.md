title:EZ2ST - TIME TRAVELER TERMINAL



concept：
EZ2STは、単なるスコア管理ツールではない。
これは、プレイの瞬間を刻むための“記録装置”である。
アーケードで初めて「自分のDJネーム」を刻んだあの日の感動。
古き良きローカルランキング文化。
それらを、もう一度“体験する”。



概要：
EZ2ONのリザルト画面を読み取り、その場でプレイヤーネームを刻み、
ランキングとして保存・表示するツールです。



特徴：
・OCRによるリザルト自動読み取り
・ドットセグメント風の腕時計UI
・名前入力による「刻まれる体験」
・軽量設計（プレイを邪魔しない）



事前準備（重要）：
本ツールを使用する前に、EZ2ON側で以下の設定を行ってください。
・リザルト画面が表示される状態で使用すること
・画面表示：仮想フルスクリーンモード
・解像度：1920*1080
・言語設定：ENGLISH
※これらの条件が一致しない場合、OCRが正しく動作しない可能性があります。



起動方法：
・main.exe を起動してください
・ゲーム起動後、画面下部にUIが表示され、本ツールは常駐します



使用方法：
リザルト画面にて
HOMEキー：長押し（約3秒）でスキャン開始
ランクイン時
↑ / ↓：文字選択
←：1文字削除
→：文字決定
ENTER：登録
ESC：キャンセル



動作確認環境：
・解像度：1920*1080
・画面表示：仮想フルスクリーン
・OS：Windows 10
